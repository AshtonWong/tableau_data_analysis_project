# PM10 Social Media & News Analysis

## Overview

This project analyzes social media and news conversations surrounding PM10 to uncover public sentiment, engagement patterns, discussion trends, and influencer activity.

Using Tableau, I transformed raw social media data into actionable insights by answering key business questions through data visualization and exploratory analysis.

## Tools Used

* Tableau
* Microsoft Excel
* Data Analysis
* Data Visualization
* Exploratory Data Analysis (EDA)
