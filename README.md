# PM10 Social Media & News Analysis

## Overview

This project analyzes social media and news conversations surrounding PM10 to uncover public sentiment, engagement patterns, discussion trends, and influencer activity.

Using Tableau, I transformed raw social media data into actionable insights by answering key business questions through data visualization and exploratory analysis.

## Tools Used

* Tableau
* Microsoft Excel
* Data Analysis
* Data Visualization
* Exploratory Data Analysis (EDA)


## Questions Answered

### When did the conversation pick up, peak, and taper off?

Analyzed conversation volume over time to identify periods of heightened public attention.

### Where did the conversation come from?

Determined which social media platforms contributed most to the discussion.

### What did the public feel about the issue?

Evaluated sentiment distribution across social media conversations.

### Who generated the most buzz?

Identified the most influential accounts based on engagement metrics.

### How many unique users talked about the issue?

Measured the overall reach and participation level of the discussion.

### What were the most commonly used hashtags?

Analyzed hashtag frequency to understand key discussion themes.
